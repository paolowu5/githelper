# Git Manager

Git Manager is a simple and intuitive tool for managing Git repositories. It streamlines daily Git operations by providing an easy-to-use command-line interface for tasks like managing branches, making commits, pushing changes, and more.

## Features

- **Branch Management:** Create, rename, and switch between branches with ease.
- **Commit and Push:** Stage files, create commits, and push changes to remote repositories.
- **Repository Status and Log:** View the current repository status and commit history.
- **Interactive CLI:** Navigate the tool using a straightforward command-line menu.
- **Remote Management:** Easily configure remote origins and handle push and pull operations.

## Installation

To get started with Git Manager, follow these steps:

1. **Clone the repository:**

   Open your terminal (bash) and clone the repository:

   `bash
git clone https://github.com/paolowu5/githelper.git`

   Navigate into the project directory:

   `cd githelper`

   Install the only dependancy required (colorama):

   `pip install -r requirements.txt`

   Run the Git Manager:

   After installing the dependencies, run the application by executing:

   `python main.py`
